IDRGP 2023

Script para o monitoramento do IDRGP no exerccio de 2023

Intoduo

Objetivo: levantar, tratar e analisar dados dos valores executados, nas regies da cidade de so paulo a partir de uma cesta de desepesas selecionadas, que incluem o Programa de Metas 2021-2024 Verso Alterao Programtica, uma lista de obras monitoradas e intervenes do Oramento Cidado

Pacotes utilizados:

library(tidyverse)
library(readxl)
library(purrr)
library(sf)

base::options(scipen = 99)

Base cartogrfica do Municpio

Em seguida, inserimos a base cartogrfica do municpio de So Paulo obtida a partir do GeoSampa.

A partir dela selecionamos os dados das Subprefeituras com os valores de referncia do ndice.

Adicionando a base de Obras Prioritrias

Para gerar a base de Obras Priotirrias, iremos abrir a base encaminhada pela Unidade de Entregas da SEPEP, padronizar nomes e realizar os filtros, como os que removem obras de mobilidade e macrodrenagem, pois estas obras no compem o clculo do IDRGP.

Ento, agregamos as observaes por Subprefeitura

Em seguida, conectamos com a base catogrfica para georreferenciar os dados

Por fim, com objetivo meramente descritivo, vamos visualizar os dados das Obras Prioritrias em dois gricos, especificando a liquidao por Secretaria, a liquidao por Subprefeitura (ainda no obtemos os dados de liquidao por Ao Oramentria).
